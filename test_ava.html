Please note: The array is dual-indexed due to the nature of the JSON we return.
It will always begin with productArray[0][n] where n is the number of the item.
Also, it is 0-indexed, meaning 0 is the first item, 1 is the second, etc.
Examples:
productArray[0][0] = 1st item = Patagonia Classic jacket
productArray[0][1] = 2nd item = North Face
productArray[0][2] = 3rd item = Arcteryx
